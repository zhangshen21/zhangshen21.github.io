    《水调歌头·黄州快哉亭赠张偓佺》 #

    苏轼 #

    落日绣帘卷，亭下水连空。知君为我新作，窗户湿青红。
    长记平山堂上，欹枕江南烟雨，杳杳没孤鸿。认得醉翁语，“山色有无中”。　

    一千顷，都镜净，倒碧峰。忽然浪起，掀舞一叶白头翁。
    堪笑兰台公子，未解庄生天籁，刚道有雌雄。一点浩然气，千里快哉风。
